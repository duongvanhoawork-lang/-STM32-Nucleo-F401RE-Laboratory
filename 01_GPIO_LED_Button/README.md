# 🟢 GPIO: LED and Button Control

## 📝 Overview
This section covers the fundamental concepts of General Purpose Input/Output (GPIO) on the STM32 Nucleo-F401RE development board. It includes practical laboratory exercises focusing on basic digital output (driving LEDs) and digital input (reading push-buttons).

## 📂 Directory Structure
- `STM32_NUCLEO_F401RE/`: STM32CubeIDE / KeilC project source code.
- `Digital_I-O.docx`: Detailed lab report and documentation for the GPIO exercises.

## 🎯 Learning Objectives
- Configure GPIO pins as outputs to control LEDs (push-pull mode).
- Configure GPIO pins as inputs to read button states (pull-up/pull-down modes).
- Understand GPIO pin states and basic logic control.
- Interface hardware components with the STM32 microcontroller.
