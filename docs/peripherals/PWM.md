# PWM — Pulse Width Modulation

> STM32F401RE · HAL Reference · [Back to README](../../README.md)

---

## Overview

PWM is generated by the Timer peripheral in **Output Compare — PWM Mode**. The duty cycle is set via the **CCR (Capture/Compare Register)**.

$$\text{Duty Cycle (\%)} = \frac{CCR}{ARR + 1} \times 100$$

$$f_{PWM} = \frac{f_{TIMER}}{(PSC + 1) \times (ARR + 1)}$$

---

## CubeMX Configuration

1. Open the relevant **TIMx** → Channel → **PWM Generation CHx**
2. Set **PSC** and **ARR** for the desired frequency
3. Set **Pulse** (CCR) for the initial duty cycle
4. Enable the TIMx clock in **Clock Configuration**

---

## Common PWM Frequencies

| Use Case | Frequency | PSC | ARR (84 MHz) |
|----------|-----------|-----|--------------|
| LED brightness | 1 kHz | 0 | 83999 |
| LED brightness | 10 kHz | 0 | 8399 |
| Servo motor | 50 Hz | 83 | 19999 |
| DC motor | 20 kHz | 0 | 4199 |

## HAL API

```c
/* Start PWM on channel */
HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);

/* Stop PWM */
HAL_TIM_PWM_Stop(&htim3, TIM_CHANNEL_1);

/* Update duty cycle at runtime */
__HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty_value);
// duty_value: 0 to ARR (inclusive)
```

## Servo Motor Control

Standard hobby servos use 50 Hz PWM with 1–2 ms pulse width:

| Angle | Pulse Width | CCR (ARR=19999, 50Hz) |
|-------|-------------|----------------------|
| 0° | 0.5 ms | 500 |
| 45° | 1.0 ms | 1000 |
| 90° | 1.5 ms | 1500 |
| 135° | 2.0 ms | 2000 |
| 180° | 2.5 ms | 2500 |

```c
/* CubeMX: TIMx, PSC=83, ARR=19999 → 50 Hz at 84 MHz */
HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);

/* Set servo to 90 degrees */
__HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, 1500);

/* Map angle 0–180 to CCR 500–2500 */
uint32_t angle_to_ccr(uint8_t angle) {
    return 500 + (angle * 2000 / 180);
}
```

## Mbed OS PWM (Project Reference)

```cpp
#include "mbed.h"

PwmOut led(D6);           // D6 = PA6 (TIM3_CH1 on Nucleo)
AnalogIn pot(A0);         // A0 = PA0 (ADC_IN0)

int main() {
    led.period_us(100);   // 10 kHz PWM frequency

    while (true) {
        float value = pot.read();   // 0.0 to 1.0
        led.write(value);           // Duty cycle = potentiometer position
        ThisThread::sleep_for(10ms);
    }
}
```

## LED Brightness Control

```c
/* Fade LED up and down */
HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);
uint32_t arr = __HAL_TIM_GET_AUTORELOAD(&htim3);

while (1) {
    for (uint32_t duty = 0; duty <= arr; duty += 10) {
        __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty);
        HAL_Delay(5);
    }
    for (uint32_t duty = arr; duty > 0; duty -= 10) {
        __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty);
        HAL_Delay(5);
    }
}
```
