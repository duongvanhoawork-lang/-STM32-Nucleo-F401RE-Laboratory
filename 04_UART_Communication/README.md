# 📡 UART Communication

## 📝 Overview
This section covers Universal Asynchronous Receiver-Transmitter (UART) communication. UART is a fundamental protocol used for serial communication between the microcontroller and a PC, or between multiple microcontrollers and peripheral modules (e.g., GPS, Bluetooth).

## 📂 Directory Structure
- `UART-Based_Servo_Motor_Control .docx`: Lab report on controlling a servo motor using UART commands simulated on Wokwi.

## 🎯 Learning Objectives
- Configure UART parameters (Baud rate, data bits, parity, stop bits).
- Transmit and receive data serially.
- Retarget `printf` for easy debugging over the serial console.
- Process incoming string commands to control hardware (e.g., servo motor).
