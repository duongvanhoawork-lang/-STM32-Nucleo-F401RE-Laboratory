# ⏱️ Timers and PWM (Pulse Width Modulation)

## 📝 Overview
This section explores the usage of hardware timers and Pulse Width Modulation (PWM) on the STM32F401RE. These features are essential for precise timing, generating periodic interrupts, and controlling actuators like servos and LEDs.

## 📂 Directory Structure
- `Exercise_Easy/`: Introductory exercises for timers and basic PWM generation.
- `Exercise_Hard/`: Advanced applications of timers and complex PWM control.
- `Simulating_PWM_Timer_and_Periodic_Interupt.docx`: Lab report detailing PWM timer and periodic interrupt simulation on Wokwi.
- `Timer_PWM_Audio_Player_RGB.docx`: Documentation on using Timers and PWM for Audio Player and RGB LED control.

## 🎯 Learning Objectives
- Configure hardware timers (TIM) for precise delays and periodic execution.
- Generate PWM signals to control LED brightness and motor speed.
- Handle timer interrupts and understand prescaler (PSC) / auto-reload (ARR) concepts.
