# ⚡ EXTI: External Interrupts

## 📝 Overview
This section demonstrates how to handle External Interrupts (EXTI) on the STM32F401RE. Using interrupts allows the microcontroller to respond immediately to external hardware events (like a button press) without the need for constant polling, improving efficiency and response time.

## 📂 Directory Structure
- `External_Interrupt_Control_Using_Push_Buttons_and_RGB_LEDs.docx`: Lab report on external interrupt control using push buttons and RGB LEDs.
- `Interrupt_and_Pull-Up_Button.docx`: Documentation on interrupt and pull-up button configurations.
- `Interrupt_Button_Counter_with_RGB_LED_Priority_and_Serial_Output.docx`: Lab report for button counter with RGB LED priority logic and serial output.

## 🎯 Learning Objectives
- Configure GPIO pins to trigger external interrupts (rising/falling edge).
- Implement Interrupt Service Routines (ISRs) to handle asynchronous events.
- Understand interrupt priorities and the NVIC (Nested Vectored Interrupt Controller).
- Solve common issues like button bouncing (debouncing techniques).
